# E-commerce Website

## Description

A fully functional e-commerce website built using Django and PostgreSQL, featuring product listings, shopping cart functionality, user accounts, and payment processing.

## Features
- Product Listing
- User Authentication
- Shopping Cart
- Payment Integration

## Technologies Used
- **Backend**: Django, Django Rest Framework, PostgreSQL
- **Frontend**: React, Bootstrap
- **Payment Processing**: Stripe or PayPal API

## How to Run Locally

### Backend
1. Clone the repository.
2. Navigate to the backend directory and set up a virtual environment.
3. Run `pip install -r requirements.txt`.
4. Update `settings.py` for database connection.
5. Run the migrations: `python manage.py migrate`.
6. Start the server with `python manage.py runserver`.

### Frontend
1. Navigate to the frontend directory.
2. Run `npm install`.
3. Start the app with `npm start`.
